1. An Overview of Natural Language Processing

1.1 Natural language understanding vs Natural language generation

  • NLU systems focus on analysing natural language
  • NLG systems focus on generating natural language
    • From natural language (chatbots): human-generated text -> response
    • From structured data (financial reporting, weather prediction): structured database -> description/narration of the data
    • From media data(generating image captions): media -> text（image caption, lipreading algorithms）

1.2 Common application of NLP

  • Automated Summarisation: producing a summary of arbitary length for a piece of text
  • Question Answering: Siri
  • Information Retrieval: search engines, recommender systems
  • Conversational User Interfaces: Chatbots

2. The NLP pipeline

  • Each unit of interest that we analyse is a document
    • A book, parapraph, tweet can be a document
  • A collection of interest of documents is a corpus
    • The Gutenberg project libray, any arbitrary subest can be a corpus

1. Tokenisation

  • A document is made of tokens
  • Tokens can be of arbitrary length
  • We care about characters(spelling correction), syllables(speech recognition), words

    2. Anontation

  • Part of speech (POS) tagging gives each word their grammatical function in a sentence
  • Different languages have different parts of speech, like English has Noun, Verb,Adjective, Adverb
  • POS tags help other algorithms work better

    3. Word standardisation: lemmatising vs stemming

  • Lemmatisation: reducing terms to their lemma (dictionary form):
    • dogs - dog
  • Stemming: reducing terms to their word stem (common part of the inflections):
    • programmer -> program
    • programming -> program

4. Filtering: stop-words

  • Some words are noise(determinants)
  • We can also filter words by frequency
    • One that only appears once or twice(not really representative)
    • One that appears everywhere (no discriminative power)

3. n-grams

  • Unigram: single word
  • Bigram: two consecutive words
  • Trigram: three consecutive words
  • 4-gram: 4 consecutive words
