Report

We apply a Neural Network as the backbone of the Face2Multi-modal. We select DenseNet as the model architecture because of its efficiency in training time, and hyperparameter adjustment. In the context of predicting the driver’s status, original settings might not be as effective as well because all the input is too similar(they are all driver’ s facial expressions with slight differences). Therefore, we apply some lightweight changes in model settings to adapt to the BROOK database. More details about the hyperparameter setting of Densenet are provided in following.

Parameters Value
Depth/Layers 100
Growth Rate 12
Dense Blocks 4
Compression Factor 0.5
Batch Size 128
Initial Learning Rate 0.1
Training Epochs 50

Value 100 12 4 0.5 128 0.1 50
Status Heart Rate Skin Conductance Vehicle Speed
Accuracy 83.78% 59.89% 58.60%

We used BROOK which is a public multi-modal database with facial video records as the dataset. The dataset contains 22 driver’s facial videos labeled with heart rate, skin conductance, and vehicle speed. We split the training set, test set, and validation set in a ratio of 8:1:1 followed the recommended settings. The PyTorch is used for the implementation of the Neural Network model.
